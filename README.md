## Установка
На Windows:
1. Скачать и запустить *-win-*.exe 
2. Следуйте стандартным инструкциям установщика

На macOS:
1. Скачать и распаковать *-mac-*.zip
2. Перетащить приложение в папку "Программы" (Applications), но НЕ запускать
3. В терминале выполнить: 
```sh
xattr -cr /Applications/actionboard_demo.app
```
4. Можно запускать
